# SLScheevo

Achievement generator for [SLSsteam](https://github.com/AceSLS/SLSsteam), originally forked from [Detanups old GBE_Tools](https://github.com/Detanup01/gbe_fork_tools/), stripped logic and automated

## Prerequisites:

Python 3 (Worked on with 3.12, but should work on older as well)

## How to run:

- Linux: run.sh or download the [latest release](https://github.com/xamionex/SLScheevo/releases/latest/download/SLScheevo-Linux.tar.gz)
- Windows: run.bat or download the [latest release](https://github.com/xamionex/SLScheevo/releases/latest/download/SLScheevo-Windows.zip)
